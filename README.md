# Meraki Sipside Coffee Shop

Welcome to the Meraki Sipside project! This website is designed to showcase our coffee shop, offering a delightful experience for coffee lovers and food enthusiasts alike.

## Project Structure

The project is organized as follows:

```
meraki-sipside
├── src
│   ├── index.html          # Main entry point of the website
│   ├── pages               # Contains individual page HTML files
│   │   ├── home.html       # Home page showcasing offerings and ambiance
│   │   ├── browse.html     # Menu browsing page
│   │   └── checkout.html    # Checkout process page
│   ├── styles              # Stylesheets for the website
│   │   ├── style.css       # Main stylesheet
│   │   ├── home.css        # Styles for the home page
│   │   ├── browse.css      # Styles for the browse page
│   │   └── checkout.css     # Styles for the checkout page
│   ├── scripts             # JavaScript files for functionality
│   │   ├── app.js          # Main JavaScript file
│   │   ├── cart.js         # Shopping cart management
│   │   └── search.js       # Menu search functionality
│   └── assets              # Directory for assets
│       └── fonts           # Font files used in the website
├── README.md               # Project documentation
└── package.json            # npm configuration file
```

## Features

- **Home Page**: A welcoming introduction to Meraki Sipside, highlighting our unique offerings and ambiance.
- **Browse Menu**: An easy-to-navigate menu for browsing our selection of coffees and snacks.
- **Checkout Process**: A streamlined checkout experience for customers to review and complete their orders.

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install the necessary dependencies using npm:
   ```
   npm install
   ```
4. Open `src/index.html` in your web browser to view the website.
